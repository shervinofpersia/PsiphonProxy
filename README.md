### ✨ درباره پروژه

**SHΞN Proxy Hub** یک هاب مدرن و قدرتمند برای دریافت پروکسی‌های تازه و باکیفیت است که مخصوص کاربران ایرانی و علاقه‌مندان به **Psiphon** طراحی شده.

این پروژه به‌طور خودکار پروکسی‌های معتبر از سرورهای مختلف دنیا جمع‌آوری کرده و به صورت مرتب به‌روزرسانی می‌شود تا همیشه بهترین گزینه‌ها در دسترس شما باشد.

### 🎯 قابلیت‌ها

- **پروکسی‌های زنده و تست‌شده** از چندین کشور مهم دنیا
- پشتیبانی از انواع پروتکل‌ها: **HTTP / HTTPS / SOCKS5**
- رابط کاربری مدرن، responsive و کاملاً گرافیکی (Glassmorphism + Cyber Style)
- فیلتر هوشمند بر اساس کشور و نوع پروکسی
- به‌روزرسانی خودکار روزانه از طریق GitHub Actions
- مناسب برای **Psiphon**، **شیروخورشید** و سایر ابزارهای ضدسانسور
- کپی آسان با یک کلیک

### 🚀 نحوه استفاده

1. به وبسایت پروژه بروید:
   🚸 **[SHΞN PROXY HUB☬](https://shervinofpersia.github.io/PsiphonProxy)**

2. کشور و نوع پروکسی مورد نظرتان را انتخاب کنید.

3. روی آدرس پروکسی کلیک کنید تا به‌طور خودکار کپی شود.

4. در **Psiphon** یا **شیروخورشید**:
   - حالت **Proxy Mode** را فعال کنید
   - پروکسی کپی‌شده را در تنظیمات وارد کنید
   - یا از حالت **Direct Connection** با پروکسی‌های باکیفیت استفاده کنید
